Your price comparison website is built using Python and Django, with web scraping powered by Beautiful Soup and Selenium. When a user enters a product name to compare prices, the backend processes the query and utilizes Selenium to scrape data from e-commerce websites like Amazon, Flipkart, Chroma, and Vijay Sales. Selenium simulates human interaction to bypass web scraping restrictions. Beautiful Soup extracts data from the websites' HTML structure, and prices are converted to floating-point numbers for comparison. The scraped products are stored in a list, which is sorted to identify the most affordable product with the lowest price. This information is then displayed on the search page, allowing users to make informed purchasing decisions by clicking on the product link to access the e-commerce site directly.
